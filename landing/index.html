Qué hace

🔒

Local o cloud, tú decides

Modelos Whisper abiertos 100% en tu Mac (gratis, offline, privado, con GPU), o una API cloud (Groq / OpenAI) con tu propia key.

🎙️

Susurros incluidos

Mantén la tecla, habla bajito, suelta. El texto sale limpio y se pega solo en la app que tengas delante.

🔑

Tu key, tu máquina

Las API keys se guardan cifradas en el Keychain de macOS. Nada sale de tu equipo salvo al proveedor que elijas.

🕘

Historial + insight

Cada dictado queda en local. Recupéralo, re-pégalo, y mira en qué apps y cuánto has dictado.

Instalar

1
Descarga el .dmg, ábrelo y arrastra la app a Aplicaciones.
O por terminal: brew install --cask netraluis/tap/netraluiswhisper
2
Primera vez (app sin firmar, macOS la bloquea):
xattr -dr com.apple.quarantine /Applications/netraluisWhisper.app
y ábrela. (Con Homebrew este paso no hace falta.)
3
Concede permisos cuando macOS pida: Micrófono, Input Monitoring y Accesibilidad.
4
Dicta. Mantén Cmd derecho, habla, suelta. Configura motor, modelo, idioma y tecla desde el icono 🎙 del menubar.
